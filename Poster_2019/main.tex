            		\item Key points (\textbf{landmarks}): the points on the image that are invariant when the image changes
             		\item \textit{Key points detection}: to find the key points through images (video)

            \begin{block}{Conclusion}
				\begin{enumerate}
					\item A CNN has been proposed to predict the landmarks on pronotum images which are difficult to apply image processing techniques.
					\item A new method has been applied to augment dataset.
					\item The quality of predicted landmarks have been evaluated by average distances with an accuracy greater than $80\%$.
					\item The predicted landmarks can be used to replace manual landmarks.
				\end{enumerate}				            
            \end{block}
